Регистрация администратора системы

Не обнаружено ни одной учётной записи в системе, необходимо создать первую учётную запись, самого первого администратора.

Не потеряйте пароль администратора системы. Пароль должен быть только у вас, так как администратор системы имеет все права. После заверщения этапа первой регистрации создайте новых администраторов с ограниченными правами.
